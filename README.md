# auto-sorter

Маленький скрипт, который раскладывает файлы в папке по подкаталогам — по типу файла или по месяцу изменения. Написал, потому что надоело руками разгребать `Downloads`.

Никаких зависимостей — только стандартная библиотека Python.

## Как использовать

```bash
# посмотреть, что будет сделано, без реальных перемещений
python sorter.py ~/Downloads --dry-run

# разложить по типу (images/documents/video/...)
python sorter.py ~/Downloads

# разложить по месяцу изменения файла
python sorter.py ~/Downloads --by date
```

Пример вывода:

```
report.pdf  ->  documents/report.pdf
photo.jpg   ->  images/photo.jpg
song.mp3    ->  audio/song.mp3

Готово: перемещено 3 файлов.
```

## Как это работает

- Расширение файла сопоставляется с категорией через словарь в `sorter.py` (images, documents, video, audio, archives, code, installers, other).
- Если сортировка по дате — берётся `mtime` файла и группировка идёт по `YYYY-MM`.
- Если файл с таким именем уже есть в целевой папке — добавляется суффикс `_1`, `_2` и так далее, ничего не перезатирается.
- Скрытые файлы (начинаются с `.`) и вложенные папки не трогаются.

## Ограничения

- Работает только с файлами верхнего уровня папки, рекурсивно внутрь не заходит — специально, чтобы не сломать чужую структуру подкаталогов.
- Список расширений можно и нужно дополнять под себя — он лежит прямо в начале `sorter.py`.
